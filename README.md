# Simplex Lab

Aplicacion estatica para la materia Investigacion Operativa.

## Que hace

- Permite definir la cantidad de variables y restricciones.
- Deja cargar la funcion objetivo y cada restriccion con coeficientes editables.
- Soporta problemas de maximizacion y minimizacion.
- Soporta restricciones `<=`, `>=` y `=`.
- Genera automaticamente las iteraciones del metodo Simplex con las tablas del tableau.
- Dibuja la region factible, el punto optimo y las restricciones vinculantes cuando el problema tiene 2 variables de decision.

## Stack

- `index.html`
- `styles.css`
- `script.js`

No requiere build, dependencias ni backend. Eso lo deja listo para publicar en GitHub Pages.

## Uso local

1. Abrir `index.html` en el navegador.
2. Definir variables y restricciones.
3. Completar coeficientes y resolver.

## Publicacion en GitHub Pages

1. Subir estos archivos al repositorio.
2. En GitHub, ir a `Settings > Pages`.
3. Elegir la rama principal y la carpeta raiz (`/root`).
4. Guardar y esperar la publicacion.
